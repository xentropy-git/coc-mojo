# coc-mojo

A coc wrapper for the Mojo LSP provided by Modular.

## Install

`:CocInstall coc-mojo`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-mojo-keymap)`

## Lists

`:CocList demo_list`
